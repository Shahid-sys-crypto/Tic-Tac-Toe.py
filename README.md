## ğŸ§  How It Works

### Game Logic

- The game uses a **3Ã—3 board** stored as a 2D list
- Players alternate turns between **X** and **O**
- Clicking a button places the current player's mark
- Buttons are disabled once the game ends

### Win Conditions

A player wins if they have:
- Three marks in a row
- Three marks in a column
- Three marks diagonally

### Draw Condition

- If all cells are filled and no winner is found, the game is declared a draw
